# Tableau recap - outils IA

Pour l'event **L'IA a la cool**

## Tableau principal

| Besoin | Outil | A quoi ca sert | Meilleur cas d'usage | Pour une utilisation basique |
|---|---|---|---|---|
| Assistant IA general | **ChatGPT / GPT-5.5** | Reflechir, ecrire, synthetiser, expliquer, creer des plans | Quand on veut un cerveau polyvalent simple a utiliser | **Oui, le meilleur point d'entree** |
| Images IA | **ChatGPT Images 2.0** | Creer ou modifier des visuels par conversation | Posts reseaux, visuels de marque, mockups, idees creatives | **Oui, tres recommande** |
| Video IA | **Seedance 2.0** | Generer des videos IA avancees | Clips courts, videos creatives, rendu premium | Non, plutot avance/creatif |
| Pub / Ads | **GoMarble** | Analyser et optimiser des campagnes publicitaires | Quand on gere Meta/Google Ads et qu'on veut ameliorer les performances | Non, utile si on fait deja de la pub |
| LLM bon rapport qualite/prix | **Kimi K2 Thinking** | Modele IA puissant et moins cher via API/OpenRouter | Gros volumes, automatisations, agents, taches longues | Non, plutot pour profils avances |
| LLM economique | **DeepSeek V4 Pro** | Alternative moins chere pour beaucoup de texte ou d'automatisation | Budget serre, gros volume, tests API | Non, sauf public tech |
| LLM premium | **Claude Opus 4.7** | Raisonnement profond, ecriture/codage haut niveau | Cas complexes ou la qualite prime sur le cout | Pas en premier choix |
| Agents IA 24/7 | **Hermes Agent sur VPS** | Agent autonome qui tourne en continu | Automatiser des taches recurrentes avec memoire | Non, necessite de la technique |
| SEO complet | **Distribb** | Articles SEO + backlinks + strategie de visibilite | Gagner du trafic organique sans tout faire a la main | Oui si l'audience a un site |
| Backlinks SEO | **Dofollowback** | Obtenir des liens dofollow | Booster l'autorite d'un site existant | Non, usage SEO specifique |

## Comparaisons simples

| Si la personne hesite entre... | Choisir plutot... | Pourquoi |
|---|---|---|
| ChatGPT vs Kimi vs DeepSeek vs Claude | **ChatGPT** | Le plus simple pour debuter, sans setup technique |
| ChatGPT Images vs Seedance | **ChatGPT Images** pour image, **Seedance** pour video | Pas le meme usage : image statique vs video |
| Distribb vs Dofollowback | **Distribb** | Plus complet : contenu + SEO + backlinks |
| Kimi vs DeepSeek | **DeepSeek** pour cout, **Kimi** pour agents/raisonnement long | Deux outils API plutot avances |
| GPT-5.5 vs Claude Opus | **GPT-5.5** pour usage general, **Claude** pour certains cas premium | Claude est plus cher, utile quand la qualite specifique justifie le cout |

## Recommandation pour l'event

Pour une audience **L'IA a la cool**, presenter d'abord le trio de base :

| Niveau | Outil conseille | Message simple |
|---|---|---|
| Debutant | **ChatGPT** | Ton assistant pour ecrire, reflechir, organiser. |
| Creatif | **ChatGPT Images 2.0** | Tu decris une image, il te la cree ou la modifie. |
| Business / site web | **Distribb** | Si tu veux etre visible sur Google sans devenir expert SEO. |

Mettre le reste en **bonus pour aller plus loin** :

- **Seedance 2.0** pour la video.
- **GoMarble** pour la pub.
- **Kimi / DeepSeek / Claude** pour les utilisateurs avances.
- **Hermes Agent** pour l'automatisation 24/7.
